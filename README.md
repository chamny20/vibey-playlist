## 🧑🏻‍💻 프로젝트 소개

Vibey는 음원 스트리밍 서비스와 SNS가 결합된

<strong>"음원 스트리밍 SNS"</strong> 입니다.

음원 사이트와 같이 내가 원하는, 또는 몰랐던 음악 청취가 가능합니다.

또한, SNS와 같이 유저 간 팔로우를 통한 소통을 할 수 있습니다.

## ⚙️ 주요 기능

### 1) 서핑

- 키워드를 입력하여 원하는 음악 검색 가능
- 음악 선택 후, 미리듣기 기능
- 플레이리스트에 추가 및 삭제 기능

### 2) 오늘의 추천

- 매일 새로운 음악 추천
- 플레이리스트에 추가 가능

### 3) 프로필

- 플레이리스트에 추가된 음악은 피드에 반영
- 음악은 추가 및 삭제 가능
- 팔로우를 통해 유저 간 소통 가능
